## Backend end Built With
- Node.js
- Express
- Mysql

(nodemon, npm, and MySQL already installed.)
## Set up the backend
Create a new database in MySQL called users.

Install server dependencies:

$ cd capstone-backend
$ npm install

### Run migrations

$ npm run migrate

### Run seeds

$ npm run seed

$ npm run dev
